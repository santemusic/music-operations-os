# Music Operations OS - 600 Prompt Library

A professional prompt library for music backoffice operations: label operations, artist management, A&R, publishing, marketing, DSP growth, booking, touring, finance, royalties, and legal operations.

## Pack Workflows

### Pack 1 - Executive & Label Operations
Daily: inbox summary -> daily label brief -> priority matrix -> executive daily plan. Weekly: operations review -> release pipeline -> invoices -> risk register -> leadership agenda. Monthly: finance/reporting -> roster -> catalogue -> cashflow -> operations improvement.

### Pack 2 - Artist Management
Daily: artist status -> priority matrix -> action extraction -> content schedule -> time management. Weekly: status review -> task delegation -> music/release review -> brand/content review -> revenue/live review -> risk/wellbeing -> 7-day plan.

### Pack 3 - A&R & Publishing
Daily: demo triage -> pipeline update -> publishing intake -> split confirmation. Weekly: evaluate priority music -> develop songs -> plan sessions -> publishing admin -> rights clearance -> sync/song pitching -> reports.

### Pack 4 - Marketing & DSP Growth
Daily: KPI dashboard -> content calendar -> TikTok/short-form push -> algorithmic signals. Weekly: analyze streaming/social/ads -> refine positioning -> DSP/playlisting -> content engine -> fan activation -> PR/influencer/ads -> optimization.

### Pack 5 - Booking & Touring
Daily: booking pipeline -> holds -> pitch/follow-up -> advance checks. Weekly: target venues/promoters -> evaluate offers -> routing/budget -> contracts/payments -> advancing -> logistics -> promo -> settlements/reporting.

### Pack 6 - Finance, Royalties & Legal
Daily: invoices -> payment approvals -> overdue follow-up -> contract summaries -> red flags. Weekly: cashflow -> payments -> royalties -> contracts -> rights/clearance -> tax/compliance docs -> risk registers.
